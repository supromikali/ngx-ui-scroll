If we are not going to deal with negative indexes, we can fix the datasource left boundary at 1 (or 0 depends on your needs). This is a particular case of limited datasource discussed above with only MIN value set up. Also, positive limitation could be alternatively achieved by assigning 1 (or 0) value to minIndex setting.

Note that if 0 index is the left boundary of the data set, then it seems reasonable to assign 0 value to startIndex setting too, because the default startIndex value is 1.
