What if we want to start the uiScroll from some specific position? It is possible via startIndex setting which determines what item index the uiScroll will use to start the load process. It will be the value of the index argument during the first Datasource.get method call.

The default startIndex value is 1. It has to be an integer with no range limitations. But real datasource boundaries (in case the Datasource is limited) should be taken into the account.
